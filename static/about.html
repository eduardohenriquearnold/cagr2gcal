Esse sistema foi criado por Eduardo Henrique Arnold, aluno de graduação de engenharia elétrica da UFSC.

A motivação foi usar a linguagem Python num problema real de forma a concretizar conceitos e desenvolver habilidades em diferentes tópicos de sistemas de informaação, diga-se, protocolo HTTP, HTML 'parsers', API's do Google e interface do usuário com Python através da Web. Já o objetivo foi possibilitar ao usuário final um processo simples e rápido para importar o seu calendário de aulas do sistema da UFSC para o serviço de calendário do Google.

Esse é um software de código livre, protegido pelos termos da licensa GNU GPLv3. Você pode encontrar os arquivos fonte aqui.

Agradecimentos à comunidade Python e contribuintes dos seguintes pacotes:

  • Flask
  • BeautifulSoup
  • google-api-client-python
