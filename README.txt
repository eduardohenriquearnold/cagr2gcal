CAGR2CAlendar
    É um projeto para importar os horários de aula dos alunos de graduação da UFSC (Universidade Federal de Santa Catarina), disponíveis no sistema de controle CAGR, para o serviço de calendário do Google, Google Calendar. Esse projeto foi feito utilizando Python 2.7, é de código livre e é protegido sob os termos da licença GPLv3.

-Pacotes Python Adicionais Necessários:
    google-python-client-api : Interface com API do Google Calendar
    beautifulsoup4 : Ferramente para ler HTML com mais facilidade
    Flask : Ferramenta para criar um ambiente WEB para o usuário final
